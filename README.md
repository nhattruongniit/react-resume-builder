# RESUME BUILDER FULLSTACK APPLICATION

This is a full-stack web application that allows users to create, customize, and download professional resumes. The application is built using modern web technologies and provides an intuitive interface for users to input their information and select from various resume templates.

## Technologies Used

### Frontend
- React
- TypeScript
- Vite
- Tailwind CSS
- Lucide React Icons
- Redux Toolkit

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- Multer
- ImageKit SDK
### AI Integration
- Google AI Gemini API for generating resume content based on user input.
- Docs: https://ai.google.dev/gemini-api/docs/openai#javascript
- Get API Key: https://aistudio.google.com/
